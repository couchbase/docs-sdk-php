[#latest-release]
== PHP SDK 4.2 Releases

We always recommend using the latest version of the SDK -- it contains all of the latest security patches and support for new and upcoming features.
All patch releases for each dot minor release should be API compatible, and safe to upgrade;
any changes to expected behavior are noted in the release notes that follow.

PHP SDK 4.2 is written to xref:compatibility.adoc#api-version[version 3.5 of the SDK API specification]
(and matching the features available in Couchbase 7.6 and earlier).

=== Version 4.2.4 (22 October 2024)

https://docs.couchbase.com/sdk-api/couchbase-php-client-4.2.4[API documentation]
| link:++https://github.com/couchbase/couchbase-php-client/compare/4.2.3...4.2.4++[Full Changelog]

Enhancements
============

* https://issues.couchbase.com/browse/PCBC-832[PCBC-832]:
Management API - Analytics Management (#177).

* Core updated to 1.0.3.
Release notes: xref:cxx-sdk:project-docs:sdk-release-notes.adoc#version-1-0-3-22-october-2024[C++ SDK 1.0.3].

Fixes
=====

* https://jira.issues.couchbase.com/browse/PCBC-1009[PCBC-1009]:
`IS_RESOURCE` has been removed from function signatures of the extension, so that they compile with 8.4.

* https://jira.issues.couchbase.com/browse/PCBC-1010[PCBC-1010]:
 implicit marking of certain parameters as nullable has now been deprecated.
Use explicit nullable types where applicable to avoid deprecation warning in PHP 8.4.

* Fixed `SearchQuery` API docs link in API reference.

 * https://issues.couchbase.com/browse/PCBC-997[PCBC-997]:
Excluded {cpp} files, tests, and development scripts, so that Packagist will not install them to the application `./vendor` directory

 * https://issues.couchbase.com/browse/PCBC-991[PCBC-991]:
Added consistency utility for testing (https://github.com/couchbase/couchbase-php-client/pull/168[#168]).

 * https://issues.couchbase.com/browse/PCBC-989[PCBC-989]:
Rollback to `cmake` 3.19 (https://github.com/couchbase/couchbase-php-client/pull/167[#167]).

 * https://issues.couchbase.com/browse/PCBC-988[PCBC-988]:
Fixed type annotation for `JsonSerializable` implementations (https://github.com/couchbase/couchbase-php-client/pull/166[#166]).

==== Enhancements

* https://issues.couchbase.com/browse/PCBC-994[PCBC-994]:
Support for base64 encoded vector types added (https://github.com/couchbase/couchbase-php-client/pull/169[#169], https://github.com/couchbase/couchbase-php-client/pull/170[#170]).
